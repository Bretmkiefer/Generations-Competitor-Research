# Claude Code Instructions

## AI Changelog Rule — Follow Every Session

This project is worked on by multiple LLMs. A shared changelog exists at `AI_CHANGELOG.md`.

**You must:**

1. **Read `AI_CHANGELOG.md` before making any edits** — understand what other LLMs have already done.
2. **After completing your changes**, add a new entry at the TOP of the Log section in `AI_CHANGELOG.md` with:
   - `### Claude Code — YYYY-MM-DD`
   - A bullet-point description of what you changed and why

This is not optional. Every session that touches files must log its changes.
